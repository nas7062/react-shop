  <h2>주요기능</h2>
  
### 회원 (User)
회원가입 / 로그인/ 로그아웃/
  
### 상품 (Item)
상품 리스트 (메인페이지, 카테고리별 페이지) / 상품 상세 페이지 / 상품 정렬

### 장바구니 (Cart)
장바구니 담기 / 장바구니 조회 / 장바구니 삭제

### 커뮤니티(comu)
커뮤니티 /상품 리뷰 /상품후기 /공지사항
